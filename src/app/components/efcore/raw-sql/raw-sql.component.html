FromSql Limitations

1. SQL queries must return entities of the same type as DbSet T> type. the specified query cannot return the Course entities If FromSql is used after Students. Returning ad-hoc types from FromSql() method is in the backlog.

2. The Sql query must return all the columns of the table context.Students.FromSql("Select StudentId, LastName from Students).ToList() will throw an exception.

3. The SQL query cannot include JION queries to get related data. Use Include method to load related entities after FromSql() method
