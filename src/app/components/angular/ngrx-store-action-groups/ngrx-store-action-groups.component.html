Action Groups


createActionGroup function tạo một nhóm các actions có cùng source. Nó chấp nhận một action group source và một event dictionary như một input đầu vào, event có kiểu key-valye (event name - event props).

{{text1}}

Nếu chúng ta tạo một action sử dụng createAction function bằng cách copy một action trước đó nhưng quên thay đổi type của nó thì vẫn pass biên dịch. Nhưng với createActionGroup function chúng ta sẽ không xảy ra trường hợp này vì chúng ta sẽ gặp lỗi biên dịch nếu 2 action trong cùng một group có cùng type.
